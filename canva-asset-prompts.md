## Export Guidelines

For ALL assets:
1. **PNG format with transparency** (high resolution: 2x or 3x for retina displays)
2. **Transparent backgrounds** (required)
3. **Optimized/compressed** (use PNG optimization tools)
4. **Consistent naming convention**: lowercase, hyphen-separated
5. **Include source files** if further editing needed
6. **Export at @2x or @3x resolution** for sharp rendering on high-DPI displays

## Color Palette Reference

Use these colors consistently across all assets:

| Color Name | Hex Code | Usage |
|------------|----------|-------|
| Deep Purple | #1a0b2e | Dark backgrounds |
| Purple | #A78BFA | Accents, borders |
| Cyan | #4ECDC4 | Highlights, links |
| Pink | #FF6B9D | Primary actions |
| Gold | #FFD700 | Premium badges |
| Green | #A8E6CF | Success states |
| White | #FFFFFF | Text, glass highlights |

## Glassmorphism Effect Parameters

Standard glass effect settings for Canva designs:

- **Blur:** 20-30px backdrop blur
- **Opacity:** 10-20% background opacity
- **Border:** 1px solid rgba(255, 255, 255, 0.2)
- **Shadow:** 0 8px 32px rgba(0, 0, 0, 0.1)
- **Inner highlight:** 1px top border rgba(255, 255, 255, 0.3)

---

**Note:** After generating, please optimize all PNGs using TinyPNG, ImageOptim, or similar tools before using in production! Export at @2x or @3x for retina displays.

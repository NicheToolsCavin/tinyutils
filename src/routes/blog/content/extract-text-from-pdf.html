Need to get text out of a PDF? Whether for analysis, editing, or reuse, here's the complete guide to PDF text extraction.

TL;DR

  • Digital PDFs: Use TinyUtils Document Converter → Plain Text
  • Scanned PDFs: Need OCR (optical character recognition)
  • Test first: Can you select text in the PDF? If yes, it's digital.

Two types of PDFs

Digital PDFs

Created from Word, HTML, or other text sources. The text is actual text data. You can select it, search it, copy it.

Scanned PDFs

Images of documents. The "text" is actually pixels. You can't select individual words. These require OCR to extract text.

How to tell the difference

  1. Open the PDF
  2. Try to select text with your cursor
  3. If you can highlight words → Digital (easy to extract)
  4. If you can only select the whole page as an image → Scanned (needs OCR)

Extracting text from digital PDFs

  1. Go to TinyUtils Document Converter
  2. Upload your PDF
  3. Select Plain Text as output
  4. Download the .txt file

Extracting text from scanned PDFs

Scanned PDFs need OCR. Options include:

  • Adobe Acrobat — Built-in OCR
  • Google Docs — Upload PDF, it runs OCR automatically
  • Online OCR tools — Various free options

Extracting tables

Table data extracts as text, but structure may be lost. For structured data:

  • Convert to DOCX first, then extract
  • Use specialized PDF-to-CSV tools
  • Manual cleanup may be needed

Common issues

Weird line breaks

PDFs store text line-by-line. Paragraphs may have breaks mid-sentence. Find-and-replace can fix this.

Missing text

Some PDFs use images for headings or styled text. Those won't extract.

Garbled characters

Custom fonts or encoding issues. Try a different extraction method or tool.

FAQ

Can I extract text from a password-protected PDF?

You need to unlock it first. Some tools can extract from read-only PDFs, but not from fully encrypted ones.

Will formatting be preserved?

Plain text has no formatting. For formatted output, convert to Markdown or DOCX instead.

Can I extract text from a specific page?

Extract the whole document, then take the portion you need.

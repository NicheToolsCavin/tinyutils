FAQ

Will CSS styling be preserved?

Inline styles convert where possible. External stylesheets and complex CSS may not transfer. Simple formatting works best.

What about JavaScript?

JavaScript is ignored — only static HTML content converts. Word documents can't run scripts.

Can I convert a whole website?

Convert individual HTML files. For multi-page sites, convert each page separately.

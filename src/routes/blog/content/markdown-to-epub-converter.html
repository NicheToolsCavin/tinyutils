Writing a book in Markdown? EPUB is the standard format for e-readers like Kindle, Kobo, and Apple Books. Here's how to convert your Markdown to EPUB.

TL;DR

  • Open TinyUtils Document Converter
  • Upload your .md file
  • Select EPUB as output
  • Download the e-book file

Why Markdown to EPUB?

  • Self-publishing — Write in Markdown, distribute as e-book
  • Documentation — Offline-readable docs for your users
  • Personal use — Read your notes on Kindle
  • Draft review — Send chapters to beta readers

What converts

  • Headings — Become chapter/section markers
  • Paragraphs — Flow naturally
  • Bold/Italic — Preserved
  • Links — Clickable in supported readers
  • Images — Embedded in the EPUB
  • Code blocks — Monospace formatting

How to convert

  1. Go to TinyUtils Document Converter
  2. Upload your Markdown file
  3. Choose EPUB from output options
  4. Click Convert
  5. Download the .epub file

Chapter structure

Use H1 headings (#) for chapters. The converter creates a navigable table of contents.

Images

Images referenced in Markdown are embedded in the EPUB. Use relative paths for local images.

FAQ

Can I read this on Kindle?

Kindle prefers .mobi or .azw3, but newer Kindles support EPUB. You can also use Calibre to convert EPUB to .mobi.

What about styling?

The EPUB uses sensible defaults. E-readers let users adjust font and size anyway.

Can I include a cover image?

Currently, covers need to be added after conversion using an EPUB editor like Sigil or Calibre.
